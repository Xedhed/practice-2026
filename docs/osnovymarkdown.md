# Основы Markdown

## 1. Заголовки
# Заголовок 1 (Обычно для названия документа)
## Заголовок 2 (Основные разделы)
### Заголовок 3 (Подразделы)

## 2. Выделение текста
*Курсив* или _Курсив_
**Жирный** или __Жирный__
***Жирный курсив***
~~Зачеркнутый текст~~

## 3. Списки
### Маркированный (обычный):
* Элемент 1
* Элемент 2
  * Подпункт (сделать 2 или 4 пробела)

### Нумерованный:
1. Первый пункт
2. Второй пункт

## 4. Ссылки и изображения
[Название ссылки](https://example.com)
![Альтернативный текст картинки](https://link-to-image.jpg)

## 5. Цитаты и код
> Это цитата. Используется для выделения важных мыслей или чужих слов.

Код внутри строки: `const x = 10;`

Блок кода (с подсветкой синтаксиса):
```javascript
function greet() {
  console.log("Hello, world!");
}
```
## Таблицы

markdown
| Заголовок 1 | Заголовок 2 |
|:-----------:|:-----------:|
| Значение 1  | Значение 2  |

Выравнивание по центру достигается с помощью двоеточий **с двух сторон** под заголовками колонок: `:---:`  
Также можно выровнять по левому (`:---`) или правому (`---:`) краю.

---

## Вывод
Markdown — это быстрый и интуитивный инструмент для форматирования. Главный плюс — текст остается легко читаемым даже в сыром виде, без компиляции.
